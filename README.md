# E-Buy Uganda

A full-featured marketplace website for buying and selling old & new items across Uganda. Built with HTML, Tailwind CSS, and JavaScript.

## Features

- **User Roles**:
  - Buyers
  - Permanent Traders (Monthly subscription)
  - Temporary Sellers (Pay after sale model)

- **Key Functionalities**:
  - Phone-based registration & 2-Factor Verification
  - MTN & Airtel Mobile Money integration
  - Direct WhatsApp seller-buyer chat
  - Product listings with images, location, and delivery options
  - Separate dashboards for Buyers, Permanent Traders, and Temporary Sellers
  - Payment history and earnings tracking
  - Public shop pages for traders
